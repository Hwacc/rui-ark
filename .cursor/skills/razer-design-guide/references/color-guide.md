# Razer 颜色指南

## 主要颜色
- **primary**: `#44d62c`
- **primary highlight**: `#73e161`
- **primary dark**: `#007200`
- **primary border** `#006b00`
- **primary border highlight** `#27c100`
- **primary border dark** `#006d00`
- **secondary** `#707070`
- **secondary highlight** `#9b9b9b`
- **secondary dark** `#4f4f4f`
- **secondary border** `#4f4f4f`
- **secondary border highlight** `#838383`
- **secondary border dark** `#373737`

## 背景颜色
- **primary background** `#111111`
- **secondary background**：`#222222`

## 其他颜色
- **razer red** `#ec3c3c`
- **razer orange** `#ffaa00`
- **razer blue** `#26a9db`

## 文字颜色
- **text black**：`#000000`
- **text black secondary**: `#373737`
- **text white**：`#ffffff`
- **text white secondary**：`#909090`

## 阴影颜色
- **shadow glory** `rgba(106,247,86,0.47)`
- **shadow black** `rgba(0,0,0,0.8)`

## 禁用状态颜色
- **disabled opacity** `50%`

## 使用指南
- primary相关颜色用于:
  - variant为primary的主色 (例如: button, checkbox等)
  - hover状态
  - 强调色
  - 焦点指示器
- secondary相关颜色用于:
  - variant为secondary或其他次要variant
  - hover状态
  - 次要强调色
  - 次要焦点指示
- primary backaground用于:
  - 主容器背景
- secondary background用于:
  - 嵌套容器
  - 提升的表面
- 文字颜色用于:
  - 内容文字的颜色
- 其他颜色用于:
  - 特殊variant配色
  - 特殊组件配色
  - 其他强调色

## 注意
- 当背景色为**priamry**相关colors或浅色背景时, 文字颜色必须为**text black**相关颜色, 进而提高对比度
- 当背景色为**secondary**相关color或深色背景时, 文字颜色必须为**text white**相关颜色, 进而提高对比度
